RunSimulation.R  -  Run all the simulation scenarios on the cluster (it calls 'CLusterFunc.R' and 'SimFunc.R')
ProcessResults.R - Process the results from 'RunSimulation.R' and generate figures
SimFunc.R - The simulation function, which implements various settings described in the manuscript
ClusterFUnc.R - Function to run the simulations in parallel on the UIUC NCSA cluster
